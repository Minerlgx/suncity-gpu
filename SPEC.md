# Suncity GPU Cloud - Specification

## Project Overview

**Name**: Suncity GPU Cloud
**Type**: Bare Metal GPU Server Rental Platform
**Target**: AI/ML deployment, LLMs, Deep Learning, Data Centers
**Language**: English
**Data Centers**: Malaysia, Indonesia, Japan (Asia-Pacific)

## Core Features

### 1. Product Pages
- GPU server listing with specs
- Multiple datacenter selection
- Hourly/Monthly pricing
- Stock status

### 2. User System
- Registration/Login
- Dashboard
- Order history
- Instance management

### 3. Admin Panel
- Order management
- User management

## UI/UX

### Visual Style
- Dark theme (professional, tech-focused)
- Blue/Purple accent colors (different from dencone's green)
- Clean, modern layout
- Emphasis on performance specs

### Color Scheme
- Primary: #3B82F6 (Blue)
- Secondary: #8B5CF6 (Purple)
- Background: #0F172A (Dark)
- Card BG: #1E293B
